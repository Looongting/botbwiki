## 🔧 常见问题

### Q: 机器人没有响应
**A**: 检查以下几点：
1. 确认 Onebot 服务正在运行
2. 检查连接地址是否正确
3. 查看日志文件 `bot.log`

### Q: 短链生成失败
**A**: 可能的原因：
1. 网络连接问题
2. 短链服务暂时不可用
3. 检索词包含特殊字符

### Q: 如何停止机器人
**A**: 在运行机器人的终端中按 `Ctrl+C`

## 📊 监控和日志

### 查看日志
```bash
# 实时查看日志
tail -f bot.log

# 查看最近日志
tail -n 100 bot.log
```

### 日志级别
- `INFO`: 一般信息
- `WARNING`: 警告信息
- `ERROR`: 错误信息
- `DEBUG`: 调试信息（需要修改配置）

## 🚀 高级配置

### 修改配置
编辑 `.env` 文件：
```bash
# 修改机器人名称
BOT_NAME=我的专属机器人

# 修改日志级别
LOG_LEVEL=DEBUG

# 修改连接地址
ONEBOT_WS_URL=ws://your-server:8080/ws
```

### 添加新功能
1. 在 `plugins/` 目录创建新的插件文件
2. 按照现有插件格式编写代码
3. 重启机器人自动加载新插件
